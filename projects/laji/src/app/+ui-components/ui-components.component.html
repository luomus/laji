Sass

How to use?

Classes are imported in angular.json and can be used globally.

Variables need to be manually imported to each sass file they are used in as such @import 'vars'. You don't need to type in the full path to the variables file, because it's configured in stylePreprocessorOptions in angular.json.

Classes

.lu-text-content

The text content class is used for styling text-heavy sections of a page. It imposes strict rules on how the html should be structured.

Following guidelines should be followed when writing text heavy html with .lu-text-content as a class of the parent element

  • The first element may be a heading with h1 title and possible p elements for descriptive text
  • Rest of the page is divided into sections
  • Headings are styled relative to section, in other words every section begins with a h1
  • Subsections and subsubsections are allowed

Alternatively to semantic html, classes can also be used for defining sections: .lu-section, .lu-subsection, .lu-subsubsection
